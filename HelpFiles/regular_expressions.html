Regular Expressions are powerful tools in computer science used in parsing text into useful information. Images can be sorted and grouped by their file name with regular expressions. If an image matches a regular expression it is grouped with others that match the expression. When wildcards are used in the grouping section of the regular expression only images with the same String in that section will be grouped together.

For example the regular expression “(a|b|c).*” checks the first letter of each file’s name and groups those that start with “a” together (ex: apple, agape, abloom), those that start with “b” (bye, by, bed), and those that start with “c” (cry, car, care). Any file that doesn’t match the expression is grouped alone (dog, rat, flat).
